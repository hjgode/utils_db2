What are or is the name of the software that is being used?

Only HSM provided software is supported.
This includes software coming with the device as:
  • Internet Explorer Mobile
  • EZConfig
  • Intermec Settings,
and software offered by HSM like
  • Enterprise Browser,
  • ITE,
  • HTML5 browser
  • Software Development Kits.

NOTE:

  • Most applications offer an About dialog with a name and version number
  • Old or outdated software is only supported in exceptions
  • For custom applications we can only support the developers in regards of using HSM SDKs or Resource Kits
