Model code and firmware/kernel version?

The model code is printed on the device (for example at battery placement)

The Intermec firmware version on actual Intermec devices can be found in:

  • Start>Settings>System>About:Packages as ITC_Version
  • in Device Health Dashboard>Information>Firmware Version
  • in Device Health log file

The Dolphin kernel version can be found  at (see also SF article):

  • On Dolphin's running Windows CE5.0. Go to: 'Start' > 'Programs' > 'Power Tools' > 'SysInfo'.
  • On Dolphins running Windows PPC, Embedded Handheld or Windows Mobile. Go to: 'Start' > 'Power Tools' > 'SysInfo'.

The LXE firmware version can be found at: (see also SF article)

  • A Low Level Test Utility is available as \Windows\utility.exe. Information-Detailed information about the terminal hardware and software configuration with Firmware Versions
