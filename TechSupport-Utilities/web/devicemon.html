deviceMon

Introduction

This is some sample code and logging tool to log device changes. These changes are also known as Plug-N-Play notifications or events.

The tool uses RequestDeviceNotifications with a MS msg queue to register for device change notifications. For example such notifications show when a device is plugged/unplugged (like a RS232 Snapon).

Installation

Just copy to the device and start using file explorer.

configuration

There is no argument to start deviceMon. The log file will created within the dir of deviceMon.exe and is called deviceMon.exe.log.txt. The log file will be backed up reaching 2MB and there is one active actual log file and one backup file.
