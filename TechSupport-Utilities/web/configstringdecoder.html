Intermec Config String V1.0.0.6

PURPOSE

Decode Intermec config (model) codes to clear text option list. Get information about the model code like WLAN, WWAN, Language, Scanner, Camera and Software options.

Installation

No installation needed. Just unpack the archive to a folder of your choice.

Usage

Just start the excecutable "ConfigStringDecoder.exe".

Enter the config (model code) into the field beside "Config String" and press ENTER. The tool will then list the options included in the model code.

You may also use the list below the "Config String" input field and select one of the available model codes.

The lower text box shows the options included for the model code.

ActiveSync / Windows Mobile Device Center support

If a device is connected via WMDC to the PC, the tool will read the config string using the USB connection. So you do not need to enter this. The recognized model code will be place in the list in the middle of the dialog.

Update

If the PC is connected to internet and can access dropbox shared sites, a click on the [Update XML] button will update the internal list of known config/model codes.
