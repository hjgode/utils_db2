Firmware / Kernel Version Dolphin

    The kernel version can also be viewed in the SysInfo file on a Dolphin. The SysInfo can be found:
  • On Dolphin's running Windows CE5.0. Go to: 'Start' > 'Programs' > 'Power Tools' > 'SysInfo'.
  • On Dolphins running Windows PPC, Embedded Handheld or Windows Mobile. Go to: 'Start' > 'Power Tools' > 'SysInfo'.
  • On Dolphins running Android follow these steps. GO to: 'All Apps' > 'Power Tools' > 'SysInfo'. The line that reads release will be the kernel version
