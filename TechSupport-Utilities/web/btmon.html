BTmon.exe

PURPOSE

Show and log Bluetooth notifications. Analyze BT connection problems.

The tool will log all Bluetooth notifications of the operating system. Used to see when and possibly why a device disconnects and how the system reacts.

INSTALLATION

Just copy BTmon.exe to your windows mobile device.

USAGE

Locate BTmon.exe and start it using the mobile file explorer. Optionally you may place a shortcut to the exe in \Windows\StartUp and so run BTmon with every reboot of the device.

The log file BTmon.exe.log2.txt is located in the same dir as BTmon.exe is located.

The log output depends on what happens in the Bluetooth environment. It will show Bluetooth notifications as Connect and Disconnect requests.
