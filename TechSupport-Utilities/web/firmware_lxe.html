Firmware / Kernel Version LXE

    A Low Level Test Utility is available as \Windows\utility.exe.
  • Information
    These give detailed information about the terminal hardware and software configuration. This is a superset of the information available in the About control panel, which is a more general user tool.
  • Firmware Versions
    This calls APIs to display the current firmware versions. This is duplicated in the About control panel.
