IntermecDevHealth

These installation files install and run Microsoft DevHealth periodically on the device. See Microsoft DevHealth description for details on what is dumped.

CN70IntermecDevhealth10min

Runs the capture every 10 minutes

Installation

Copy "CN70Intermec_Devhealth_10min.cab" onto the device and install it using the mobile file explorer

Usage

Start the periodic capture using "Start">"Programs">"DevHealthRunAt"

The captured memory dumps are located on the first external volume (normally "\Flash File Store") and are name mem<xyz>.txt.

Devhealth Rapitools

  • Batch scripts to execute DevHealth via ActiveSync and RAPI tools

Installation

Extract all files of "Devhealth Rapitools.zip" into a directory of the Windows PC that is connected (ActiveSync or Windows Mobile Device Center) to the device.

Usage

  • Ensure that the device is connected to the Windows PC.
  • Open a command prompt on the Windows. Change to the installation directory.
  • Run "DownloadandRundevhealth.bat" to copy MS DevHealth to the device and run it once.
  • Run "rundevhealth.bat" to execute MS DevHealth on the device. Use this after MS DevHealth has been copied to the device.
  • Run "Getdevhealth.bat" to copy the memory dump files of MS DevHealth to the Windows PC.
