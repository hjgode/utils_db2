BatteryLogV2

PURPOSE

Analyze power consumption of a device over time. Answer to questions like 'How long will the battery of the device last when used by a customer in the company's environment'

INSTALLATION

Just copy and install BatteryLogV2.CAB. The device will reboot after installation.

USAGE

After reboot, BatteryLog is started via a link in StartUp.

INFORMATION

The log file in the root dir of the device will have a line with battery data every 60 seconds (see configuration. The name is buld off the model code and serial number and ends on .csv:

SAMPLE

Time,AC_Line,PowerBatteryState,BatteryLifePercent,BatteryVoltage,BatteryCurrent,BatteryTemperature,PowerBatteryStrength,WiFiStatePowerOn,WiFiStateConnected,BluetoothStatePowerOn,PhoneRadioOn,PhoneSignalStrength,PowerBatteryBackupState,PowerBatteryBackupStrength,BackupBatteryLifePercent,
2015-03-12 14:46:09Z,AC_LINE_ONLINE,Charging,100,4132,0,32,VeryHigh,False,False,False,False,0,Charging,VeryHigh,99,

The first line holds the column names and all other lines the battery state for every logged time.

If the csv file can not be loaded correctly into excel insert "sep=," (without quotes) as first line using an ASCII editor like notepad.

CONFIGURATION

Registry "HKLM\Software\Intermec\BatteryLog"

  • MaxLogSizeInBytes
    • Max Log (5MB x2 default)
  • DelayInMS
    • Time between captures (60000ms default)
