Category WWAN

WWAN connections like GSM, CDMA and similar may fail for different reasons. The OS mostly shows very generic error messages. To be able to analyze WWAN issues in more details, there are logging tools like RBLogger or WWANCollector. These tools record the driver functions and messages on the device side for further analysis.
