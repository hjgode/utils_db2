Intermec_Memuse.CAB

Memuse logs the memory usage of running processes periodically and is useful to analyze devices slow downs, application crashes and similar that occur after some time. 

Installation

IntermecMemuse.CAB will install memuse.exe and links to the tool in \Windows\StartUp. As IntermecMemuse.CAB does a reboot after install I would install this after other tools that do no reboot. When the device starts the logging should start automatically.

Log File

Memuse.exe creates a log file memusage.log.txt.

Sample

The log file states the date and time of a snapshot and the processes, the process memory usage, there process ID and finally the overall memory usage.
